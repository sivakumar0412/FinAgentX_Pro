## ✨ Features
- AI-driven trading agent  
- Self-learning with rewards  
- Real-time & historical market data  
- Automated trade execution  
- Risk-aware strategy optimization  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **AI:** TensorFlow / PyTorch, Stable-Baselines3  
- **Data:** yFinance API  
- **Backend:** Flask / FastAPI  
- **Frontend:** Streamlit  
- **DevOps:** Docker, GitHub Actions  

---

## 🔁 How It Works
1. Fetch market data  
2. Preprocess input features  
3. DRL agent selects action (Buy/Sell/Hold)  
4. Trade is executed  
5. Reward is calculated  
6. Agent updates policy
